# Metro Connectivity

Second Project for Analysis and Synthesis of Algorithms

## Project Overview

This project develops an algorithm for analyzing a metro network’s connectivity, treating it as an undirected multigraph. It computes the metro connectivity index, `mc = max{ML(e1,e2) ∣ e1, e2 ∈ Metro}`, which is the largest number of line transfers needed to move between any two stations. The method deals with multiple lines, isolated stations, and varying network complexity, producing correct and efficient results. The output reports the connectivity index or special cases such as zero line changes or disconnected stations.
